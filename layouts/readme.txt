Follow these steps if you want to share a layout with the team working on this project:

1) Export the layout using the "Window" -> "Export Layout" action
2) Copy the exported layout file in the "layouts" parent folder
3) Go to "Preferences" -> "Application Layout" option page and specify the path to the layout file in "Select application layout" -> "Custom filed".
 You must use the project directory variable in the layout path: "${pd}/layouts/layout_file_name.layout"
4) Switch the option page level from "Global Options" to "Project Options"
